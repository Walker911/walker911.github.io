Cmder介绍

2018-08-10
工具
cmder

cmder 一款可以用来替代 Windows 自带 cmd 的工具。

官网：cmder，完整版也对git进行了集成。

安装

下载后，解压即可使用

版本

  • version: 1.3.6

配置

环境变量

此电脑 > 右键属性 > 高级系统设置 > 环境变量 > 系统变量，在path中添加cmder路径

中文乱码

在Settings-Startup-Environment-Set up environment variables 里面添加代码：

set LANG=zh_CN.UTF-8
set LC_ALL=zh_CN.UTF-8

IDEA集成Cmder

IDEA 开发工具支持将默认的cmd terminal 代换为 cmder，替换方法如下：

  1. 在IDEA选择File | Settings | Tools | Terminal

  2. 然后在右侧找到Application settings中的Shell path

  3. 将Shell path修改为

    "cmd.exe" /k ""%CMDER_ROOT%\vendor\init.bat""

    注意：

    • 引号不能省略
    • cmd.exe /k xx代表执行xx命令后不关闭窗口
    • %CMDER_ROOR%为配置的环境变量
